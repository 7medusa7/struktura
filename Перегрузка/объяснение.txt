В данной работе иллюстрируются основные концепции объектно-ориентированного программирования на языке C++, такие как классы, конструкторы, методы, и инкапсуляция.

Класс MyClass:

Содержит приватное поле value, которое доступно только внутри класса.
Определяет два конструктора: конструктор по умолчанию и конструктор с параметром.
Содержит два метода: getValue() для получения значения поля value (геттер) и setValue() для установки значения поля value (сеттер).
Функция main():

Создаёт два объекта класса MyClass, obj1 и obj2, используя разные конструкторы.
Выводит значения объектов obj1 и obj2, используя метод getValue().
Меняет значение поля value объекта obj1, используя метод setValue().
Выводит новое значение поля value объекта obj1.
Таким образом, код создаёт два объекта класса MyClass, иллюстрирует как можно использовать конструкторы для инициализации объектов с разными значениями, и демонстрирует работу геттеров и сеттеров для доступа к приватным полям класса.
